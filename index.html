機能
Mod管理のすべてを自動化する
インストールして5分でスキャン完了。あとはNeverLatestが自動でやります。
自動Mod検出
ゲームのフォルダをスキャンし、導入済みのプラグインやDLL系Modを自動で一覧化。バージョン情報を取得して管理対象として登録します。
一括アップデート
更新可能なModをワンクリックで全件更新。個別に選択して更新することも可能です。ゲーム起動前の自動更新スケジュールにも対応します。
更新前自動バックアップ
更新を適用する前に現在のMod構成を自動バックアップ。問題が起きたらワンクリックで更新前の状態に完全ロールバックできます。
競合チェック
同じファイルを書き換えるModを自動検出し、競合の可能性を警告。ロードオーダーの問題も提案付きで表示します。
スケジュール更新
「毎朝6時」「ゲーム起動5分前」など好きなタイミングで自動更新を設定可能。PCがアイドル中だけ動かすオプション付き。
プロファイル管理
「軽量構成」「重量Modあり」など複数のMod構成をプロファイルとして保存し、ワンクリックで切り替えられます。

FAQ
既存のMod管理ツール（Vortex / MO2）と何が違いますか？
Vortex や MO2 はModのインストール・有効化・ロードオーダー管理が主な用途です。NeverLatest は「バージョン管理と更新の自動化」に特化しており、既存ツールと併用できます。Vortex / MO2 が管理するフォルダをそのまま読み取って更新チェックを行うため、乗り換え不要で今すぐ追加できます。
更新でゲームやセーブデータが壊れることはありますか？
更新前に必ず現在のMod構成をバックアップします（デフォルトで有効、最大3世代保持）。問題が起きた場合はロールバック機能で即座に元の状態に戻せます。ただし、Mod自体の互換性の問題はNeverLatest では防げないため、大型Modの更新時は事前に情報を確認することを推奨します。
どうやってModのバージョンを管理していますか？
各Modのmanifest / metadata ファイル、DLLのバージョン情報、ファイルのハッシュ値などを読み取り、バージョンを識別します。Modが対応する情報を持っていない場合は、ファイルの更新日時とサイズを補助情報として使用します。未知のModでも可能な限り検出を試みます。
インターネット接続は必要ですか？
最新バージョンの確認と更新ファイルのダウンロードにはインターネット接続が必要です。ただし、既にダウンロード済みの更新の適用・バックアップ・ロールバックなどのローカル操作はオフラインでも動作します。
自動更新でゲームが起動しなくなった場合は？
NeverLatest の「更新ログ」から問題の更新を選び「ロールバック」ボタンを押すだけで元の状態に戻ります。解決しない場合はサポートへご連絡ください。

プライバシーポリシー

最終更新: 2025年11月18日

1. 収集する情報

NeverLatest が収集する情報は、設定で許可した場合の匿名化された利用統計（更新回数・使用ゲーム数など）のみです。ゲームデータ・セーブデータ・Modの内容を収集することはありません。アカウント登録は不要です。

2. ネットワーク通信

バージョン確認と更新ファイルのダウンロードのみに使用します。送信されるのはModのID・現在のバージョン番号のみで、ファイル内容や個人情報は一切送信しません。

3. ローカルデータ

スキャン結果・バックアップ・プロファイルはすべてローカルに保存されます。アプリをアンインストールすればすべてのローカルデータを削除できます。

4. お問い合わせ

privacy@modsync.app までご連絡ください。

利用規約

最終更新: 2025年11月18日

1. ライセンス

NeverLatest は MIT ライセンスのオープンソースソフトウェアです。個人・商用を問わず自由に使用・改変・再配布できます。

2. 免責事項

Modの更新によるゲームデータやセーブデータへの影響について、NeverLatest は一切の責任を負いません。重要なデータは事前にバックアップを取ることを強く推奨します。

3. 各ゲーム・Modの規約

各Modの利用規約はMod作者に帰属します。再配布・改変を行う場合は各Mod作者の規約を遵守してください。
