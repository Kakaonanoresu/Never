機能
Mod管理のすべてを自動化する
インストールして5分でスキャン完了。あとはModSyncが自動でやります。
自動Mod検出
ゲームのインストールフォルダをスキャンし、導入済みModを自動で一覧化。Nexus Mods・CurseForge・GitHub Release に対応し、出所不明のローカルModも識別します。
一括アップデート
更新可能なModを一覧表示し、ワンクリックで全件更新。個別に選んで更新することも可能です。スケジュール設定でゲーム起動前の自動更新にも対応します。
更新前自動バックアップ
更新を適用する前に、現在のMod構成を自動でバックアップ。問題が起きたら、ワンクリックで更新前の状態に完全ロールバックできます。
競合チェック
同じファイルを書き換えるModを自動検出し、競合の可能性を警告。ロードオーダーの問題も提案付きで表示するので、何が原因かすぐにわかります。
スケジュール更新
「毎朝6時」「ゲーム起動5分前」など、好きなタイミングでの自動更新を設定可能。PC がアイドル中だけ動かすオプション付き。
プロファイル管理
「バニラ」「重量Modあり」「軽量」など複数のMod構成をプロファイルとして保存。用途に合わせてワンクリックで切り替えられます。

FAQ
既存のMod管理ツール（Vortex / MO2）と何が違いますか？
Vortex や MO2 はModのインストール・有効化・ロードオーダー管理が中心ですが、ModSync は「更新の検出と適用」に特化しています。既存ツールと併用できるよう設計されており、Vortex / MO2 が管理するフォルダをそのまま読み取って更新チェックを行います。ModSync に乗り換える必要はありません。
更新によってセーブデータが壊れることはありませんか？
更新前に必ず現在のMod構成をバックアップします（デフォルトで有効）。万が一問題が起きた場合はロールバック機能で更新前の状態に戻せます。ただし、Modの更新によるセーブデータへの影響はMod自体の互換性に依存するため、大型Modの更新時はご自身でも事前に情報を確認することをお勧めします。
NexusモーズのAPIキーが必要ですか？
Nexus Mods からの更新確認にはNexus ModsのAPIキーが必要です（無料アカウントで取得可能）。初回セットアップ時にModSyncが案内します。CurseForge・GitHub Release については追加のAPIキーは不要です。
Steam Workshopには対応していますか？
Steam Workshop はSteamクライアント自体が自動更新を行うため、ModSync では検出・表示のみ対応しています（更新の適用はSteamに任せる形）。Nexus Mods・CurseForge・GitHub から直接ダウンロードするタイプのModが主な対象です。
自動更新でゲームが壊れた場合はどうすればいいですか？
ModSync → 更新履歴 から対象の更新を選び、「ロールバック」ボタンを押すだけで元の状態に戻ります。バックアップはデフォルトで3世代（Proは無制限）保持されます。万が一バックアップが見つからない場合はサポートまでご連絡ください。

プライバシーポリシー

最終更新: 2025年11月18日

1. 収集する情報

ModSync が収集する情報は以下に限られます。アカウント登録時のメールアドレス（Proプランのみ）、および設定で許可した場合の匿名化された利用統計（インストールMod数・更新回数など）のみです。ゲームデータ・セーブデータ・Modの内容を収集することはありません。

2. Mod情報の取り扱い

更新チェックのために Nexus Mods / CurseForge / GitHub の API を使用します。これらのAPIへのリクエストにはMod IDと現在のバージョン番号のみが含まれます。ファイルの内容や個人情報は送信されません。

3. ローカルデータ

スキャン結果・バックアップ・プロファイルはすべてローカルに保存されます。クラウドへのアップロードは行いません。アプリをアンインストールすれば、すべてのローカルデータを削除できます。

4. お問い合わせ

privacy@modsync.app までご連絡ください。

利用規約

最終更新: 2025年11月18日

1. ライセンス

ModSync は MIT ライセンスのオープンソースソフトウェアです。個人・商用を問わず自由に使用・改変・再配布できます。

2. 免責事項

Modの更新によるゲームデータやセーブデータへの影響について、ModSync は一切の責任を負いません。重要なデータは事前にバックアップを取ることを強く推奨します。

3. 各ゲーム・Modの規約について

ModSync は各Modの利用規約に従って更新を行います。Modの再配布・改変を行う場合は各Mod作者の規約を遵守してください。

4. Proプランのサブスクリプション

Proプランは月額課金です。キャンセルはいつでも可能で、次回更新日まで機能を利用できます。
